Package gnu.xml.libxmlj.transform

A JAXP-compliant wrapper for the XSLT C library for Gnome, also known as libxslt.
GnomeTransformer An implementation of javax.xml.transform.Transformer which performs XSLT transformation using libxslt.
GnomeTransformerFactory An implementation of TransformerFactory producing Transformer objects which use libxslt for transformation.
TransformTest Simple test class with command line interface.
