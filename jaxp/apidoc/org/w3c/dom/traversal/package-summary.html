Package org.w3c.dom.traversal

Supports the optional "Traversal" feature of the DOM Level 2 Recommendation.
DocumentTraversalDocumentTraversal contains methods that create NodeIterators and TreeWalkers to traverse a node and its children in document order (depth first, pre-order traversal, which is equivalent to the order in which the start tags occur in the text representation of the document).
